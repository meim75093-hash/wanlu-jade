# AI 资源规划与任务分配

目标：围绕 Wanlu Jade 翡翠原石、半明料、改口料、高端色料销售，合理分配 Codex、Hermes、OpenClaw、本地模型的任务。

## 一、当前资源

### Codex

定位：产品与工程负责人。

适合：
- 网站体验与代码开发
- 技术架构
- 转化路径设计
- 最终验收
- 复杂任务拆解
- 手机端即时指令与移动端页面验收

不适合：
- 长期无人值守发布
- 大批量重复日报

当前状态：
- Codex 手机端已连接
- 可用于手机端真实预览反馈、即时任务下发、移动端体验验收

### Hermes

当前能力：4-6 级别模型，可直接调用。

定位：运营策略、部署运维、安全管理、内容复核。

适合：
- 服务器部署
- Docker / Nginx / HTTPS
- 安全巡检
- 竞品分析
- 内容计划
- 账号运营方案
- 每日脚本复核

不适合：
- 未经确认直接操作高价值账号
- 直接处理未授权付款、交易、账号风控事项

### OpenClaw

当前状态：
- Gateway 正常运行
- Node service 正常运行
- Feishu 通道已配置
- 当前默认模型：`4sapi/claude-haiku-4-5-20251001`
- 可用更强模型：
  - `4sapi/claude-sonnet-4-6`
  - `bailian/qwen3.6-plus`
  - `hk-ticketpro/gpt-5.5`

定位：长期自动化、消息通道、定时任务、执行队列。

适合：
- Feishu 日报推送
- 定时提醒
- 发布任务排期
- 素材检查清单
- 评论/私信整理
- 运营数据表更新
- 低风险自动化

不适合：
- 默认 Haiku 模型直接管理高风险平台账号
- 无人工确认直接发布敏感内容
- 无审批删除数据、改账号权限、改支付信息

### 本地模型

定位：低成本批处理、敏感资料处理、草稿生成。

适合：
- 批量标题草稿
- 素材分类
- 视频字幕初稿
- 原石资料整理
- 内部数据摘要

不适合：
- 最终品牌文案
- 复杂商业判断
- 对外正式发布内容

## 三、OpenClaw 适合的具体任务

### P0：马上可用

1. 每晚 21:00 生成运营日报提醒
2. 每天早上 09:00 推送当天内容任务
3. 维护素材拍摄清单
4. 维护客户线索字段检查
5. 提醒 Hermes 生成第二天选题

### P1：配置后可用

1. Feishu 表格数据同步
2. Telegram/LINE/WhatsApp 私域线索归类
3. 多账号发布日历
4. 自动汇总评论关键词
5. 将高意向客户标记给人工

### P2：谨慎开放

1. 自动发布短视频
2. 自动回复客户私信
3. 自动修改账号资料
4. 自动删除/隐藏评论

P2 必须有白名单、审批和日志。

## 四、OpenClaw 模型使用建议

### 默认 Haiku

适合：
- 提醒
- 摘要
- 简单归类
- 低风险日报
- 表格字段检查

不适合：
- 平台账号安全判断
- 敏感客户沟通
- 高价值销售话术
- 自动发布最终内容

### Sonnet 4-6

适合：
- 内容复核
- 私域话术优化
- 复杂日报
- 竞品分析
- 平台风控判断

建议：OpenClaw 里涉及外部客户和账号操作的任务，至少使用 Sonnet 4-6。

### Qwen 3.6 Plus

适合：
- 中文内容批量生成
- 小红书/视频号文案
- 中文客户问题整理
- 中文销售话术初稿

### GPT-5.5

适合：
- 高级策略
- 多平台计划
- 复杂自动化规划
- 关键文档最终复核

## 五、安全边界

OpenClaw 可以管理“任务流”，但不应在初期直接全自动控制平台账号。

必须人工确认的动作：

- 发布公开视频
- 群发私域消息
- 修改账号资料
- 删除内容或评论
- 回复高价值客户
- 涉及价格、付款、保留、发货承诺

可以自动执行的动作：

- 生成草稿
- 生成提醒
- 生成日报
- 素材检查
- 评论关键词归类
- 线索标记

## 六、建议启动的 OpenClaw 自动化

### 1. 每日内容任务提醒

时间：每天 09:00  
内容：
- 今日要拍的料子
- 今日要发的平台
- 今日要补的素材角度
- 今日重点私域跟进

### 2. 每晚运营日报

时间：每天 21:30  
内容：
- 发布数量
- 播放量
- 评论数
- 私域新增
- 有效询价
- 成交/预约情况
- 明日建议

### 3. 素材缺口提醒

时间：每天 17:00  
内容：
- 哪块料缺自然光
- 哪块料缺压灯
- 哪块料缺侧边
- 哪块料缺风险说明
