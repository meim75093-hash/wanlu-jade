# Wanlu Jade 引流计划与平台规划

目标业务：销售缅甸翡翠原石、半明料、改口料、高端色料。  
核心转化：公开视频建立信任，私域一对一看货，确认风险后成交。

## 一、总目标

第一阶段不是追求大而空的曝光，而是建立稳定的“看货客户池”。

- 30 天建立 4 个主要内容账号
- 每天稳定产出 3-6 条短内容
- 每天导入 10-30 个有效私域触点
- 建立 Telegram / LINE / WhatsApp 私域承接
- 形成原石视频证据链：自然光、打灯、开窗、侧面、切料复盘

## 二、平台分工

| 平台 | 作用 | 内容重点 | 优先级 |
| --- | --- | --- | --- |
| TikTok | 海外流量入口 | 开窗、打灯、切料、市场现场 | P0 |
| YouTube Shorts | 搜索与长期沉淀 | 场口知识、避坑、切料复盘 | P0 |
| Instagram Reels | 高端品牌质感 | 高货色料、东方审美、客户案例 | P1 |
| Telegram | 私域频道 | 上新、直播预告、视频看货、成交复盘 | P0 |

| WhatsApp | 海外华人直连 | 高意向客户私聊成交 | P0 |
| 小红书/视频号 | 中文市场备用 | 科普、避坑、真实案例 | P2 |

## 三、账号矩阵

第一阶段建议先做 4 个账号，不要一开始铺太散。

1. `Wanlu Jade Source`
   - 方向：源头市场、原石实拍、直播预告
   - 平台：TikTok / YouTube Shorts

2. `Wanlu Jade Cut`
   - 方向：开窗、打灯、切石、切涨切垮复盘
   - 平台：TikTok / Reels / Shorts

3. `Wanlu Jade Academy`
   - 方向：场口、种水、色带、裂纹、避坑
   - 平台：YouTube Shorts / 小红书 / 视频号

4. `Wanlu Jade VIP`
   - 方向：上新、私域看货、客户预约
   - 平台：Telegram / LINE / WhatsApp

## 四、内容栏目

### 1. 开窗看肉

- 中文标题：这块料开窗后，肉够不够细？
- 英文标题：Window view: is the jadeite texture fine enough?
- 目的：让客户看色、肉、棉、裂。

### 2. 打灯看底

- 中文标题：压灯一圈，水头和裂都藏不住
- 英文标题：Light test reveals water, cotton, and cracks.
- 目的：建立专业信任。

### 3. 切前商量

- 中文标题：这块料先别急着切，先看这条裂
- 英文标题：Before cutting, check this risk line first.
- 目的：体现不硬卖，先讲风险。

### 4. 切后复盘

- 中文标题：这刀切得值不值？我们直接看结果
- 英文标题：Was this cut worth it? Here is the result.
- 目的：做长期信任资产。

### 5. 源头市场

- 中文标题：今天瑞丽市场看到的一块黑皮料
- 英文标题：A black-skin rough stone from today’s market.
- 目的：强化源头真实感。

### 6. 私域上新

- 中文标题：今晚可视频看货，先发自然光
- 英文标题：Available for video review tonight.
- 目的：导入 WhatsApp / LINE / Telegram。

## 五、30 天执行节奏

### 第 1 周：建立基础信任

- 每天发布：
  - TikTok 1-2 条
  - YouTube Shorts 1 条
  - Telegram 2-3 条
- 重点内容：
  - 原石实拍
  - 打灯视频
  - 看货风险说明
- 目标：
  - 完成账号包装
  - 建立 30 条基础素材库
  - 私域沉淀第一批高意向客户

### 第 2 周：强化栏目

- 固定 4 个栏目：
  - 开窗看肉
  - 打灯看底
  - 切前商量
  - 切后复盘
- 每条视频都带一个私域动作：
  - “要原视频发 WhatsApp”
  - “今晚可 LINE 视频看货”
  - “Telegram 频道先发上新”

### 第 3 周：直播与预约

- 每周至少 2 场直播或准直播素材
- 每天在 Telegram 发：
  - 当日可看货料子
  - 风险提醒
  - 预约时间
- 开始记录客户问题，整理成 FAQ。

### 第 4 周：复盘与放大

- 找出播放、评论、私信表现最好的 5 类内容
- 建立可复制模板
- 将成交案例改成：
  - 短视频
  - Telegram 图文
  - 网站案例模块
- 准备第二个月的账号扩展。

## 六、每日 SOP

上午：
- 选 3 块料子
- 拍自然光、压灯、侧面
- 生成当天脚本和标题

中午：
- 发布 TikTok / Shorts
- Telegram 发上新预告
- 回复评论，筛选询价客户

下午：
- 补拍客户要求的角度
- 整理客户问题
- 私域一对一跟进

晚上：
- 发直播/看货预告
- 做当日成交和询盘记录
- 复盘播放、评论、私信、加群数据

## 七、转化路径

```text
短视频看到料子
  -> 评论/私信问价
  -> 引导到 WhatsApp / LINE / Telegram
  -> 发自然光、压灯、侧边视频
  -> 说明场口、优势、风险、建议切法
  -> 客户确认风险
  -> 私域成交
  -> 切料复盘或客户反馈
```

## 八、私域话术

### 评论区

```text
这块可以发原视频，你要看自然光还是压灯？
```

```text
这块有风险点，不建议只看图片。要的话我发你侧面视频。
```

### 私信

```text
这块料我先发你三段视频：自然光、压灯、侧边。你看完我们再聊价格，不急。
```

```text
这块优点是水头不错，风险是侧边有暗裂，需要你确认能不能接受。
```

### 频道上新

```text
今晚可视频看货。先发自然光和压灯，不承诺切涨，风险先讲清楚。
```

## 九、素材清单

每块重点料至少准备：

- 自然光视频 1 条
- 压灯视频 1 条
- 侧边巡检视频 1 条
- 开窗/切面微距 3-5 张
- 重量、场口、皮壳说明
- 风险点说明
- 建议切法
- 是否适合直播看货
